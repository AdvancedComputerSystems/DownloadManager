    COMMANDS

    Commands buttons let Operator do simple actions on checked files. Operations are allowed just for files still in active state of download.
     
     
    In following table will be explained function of every single button:
    Button
    Function
    This form let Operator insert a Product Url and start download file
    This button let operator to access popup login and configuration panel (see below).
    This button let operator to clear the list of terminated downloads .
    This is used to purge the database. It is a convenient option for testing purposes, where it is desired to remove the previously downloaded files. This button let operator  to reset the application statistics.
    This button will open a window that explain how to edit configuration
    Pause download of checked file.
    This button will resume download of previously paused checked file.
    This button will retry download of previously stopped or error checked file.
    Cancel  will delete checked file from the queue list.
    Change positionpriority button let Operator to move on top or bottom of the list files that are still in waiting state.
