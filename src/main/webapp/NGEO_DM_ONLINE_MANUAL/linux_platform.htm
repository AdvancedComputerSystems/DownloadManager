LINUX PLATFORM

NGEO INSTALLATION PROCEDURE
 
 
 
Identifier
NGEO-CTRL-INST-0010
Title
Component Installation
Procedure Objective
 
 
 
Install and configure the NGEO component.
 
 
 
 
Files needed
 
 
 
<linux_installer>         
 
 
 
Step
Action
Expected output
 
Step-10
Login as root or as a different user (preferably root)
 
 
Step-20
cd to <installation_kit_root_dir>, i.e. a chosen directory where to copy the installed file and copy there the <linux_installer> file.
The installation package is copied.
 
Step-30
Set execute permissions to the installation file:
chmod +x <linux_installer>
The file has execute permissions.
 
Step-40
Run the following command
./<linux_installer>
An installation wizard is executed.
 
Step-50
The license agreement is shown. Press Enter to display the rest of the license. When “Do you accept the license” is shown, enter “y” and press Enter to continue with the installation.
 
 
 
 
 
Step-60
Enter the configuration properties:
·     Installation directory,
·     Repository directory,
·     DM port
·     Maximum concurrent downloads.
The default settings applied if no modification is entered are:
·     Installation directory: $HOME/<installer>
·     Repository directory: $HOME/<installer>/Repository
·     DM port: 8080
 
 
 
 
 
 
 
 
 
Step-65
Enter Y to modify the default Advanced Settings
Default Advanced Settings are:
·     Aria Port: 6800
·     CLI Username/Password : empty
Empty Username/Password means no-username and password will be required to access the HMI and the CLI
 
 
 
 
 
Step-70
Enter the SSO username and password.
 
 
 
 
 
Step-80
Click Enter to complete the installation.
The Download manager is installed in the machine.
