FILES TABLE

The Queue list offers different options for customize search, organize and apply commands on files.
 
Live Search
On the top of the tableProduct Name row,  Operator can find a search box, to simplify viewing list.
 
When entering a given string, all the rows in the files table below will be filtered, so that only rows where there is at least one field matching the string will be displayed. Operator need just to type name and press enter to start the query.
 
 
Select file
At left of the table Operator will find the column relative to the selection of files.
